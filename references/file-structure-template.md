# 业务文件结构生成模板

## 核心规则
**生成的新业务文件必须严格匹配目标文件夹中的对应业务文档的格式、结构和风格。**

## 格式分析记录
在生成文件前，先记录目标文件夹中文档的格式特征：

| 特征项 | 目标文件夹中的文档特征 | 新文件必须遵循 |
|--------|----------------------|---------------|
| 文档类型 | {original_doc_type} | {new_doc_type}（必须一致） |
| 文件扩展名 | {original_extension} | {new_extension}（必须一致） |
| 排版风格 | {original_layout} | {new_layout}（必须一致） |
| 语言风格 | {original_tone} | {new_tone}（必须一致） |
| 章节结构 | {original_structure} | {new_structure}（必须一致） |
| 编号方式 | {original_numbering} | {new_numbering}（必须一致） |
| 术语体系 | {original_terminology} | {new_terminology}（必须一致） |
| 表格样式 | {original_table_style} | {new_table_style}（必须一致） |
| 审批格式 | {original_approval_format} | {new_approval_format}（必须一致） |

## 文件清单（按目标文件夹中的分类方式组织）
```
{new_business_name}文档/
├── 制度类/
│   ├── {file_1}.{ext}    # {description}，格式同目标文件夹中的"{reference_file}"

├── 流程类/

├── 表格类/

```

## 文件生成示例（按目标文件夹中的文档格式）

### 示例 1：如果目标文件夹中有正式公文式 Word 文档
**目标文件夹中的参考文档**：`供应商管理办法.docx`
- 格式特征：正式公文、章节式（第一章/第一条）、含审批权限表、页眉页脚带公司名

**新文件**：`渠道商管理办法`（完全相同的格式和风格）
```

第二条 适用范围
  本办法适用于公司所有渠道商的准入、考核、退出管理。

第二章 渠道商准入

[待填写：考核指标、退出条件]

|----------|--------|----------|

```

### 示例 2：如果目标文件夹中有步骤式 Markdown 流程文档
**目标文件夹中的参考文档**：`费用报销流程.md`
- 格式特征：Markdown、步骤式编号、含角色职责、产出物标记

**新文件**：`差旅报销流程.md`（完全相同的格式和风格）
```markdown

- **产出物**：《出差申请表》

- **负责角色**：直属上级
- **操作内容**：审核出差必要性及预算合理性
- **产出物**：审批意见
- **时限**：1 个工作日

### 步骤 3：[待填写]
...

## 三、注意事项
- [待填写：特殊情况处理]
```

## 生成检查清单
- [ ] 所有文件扩展名与目标文件夹中的同类文档一致
- [ ] 文档结构（章节层级、标题样式）与目标文件夹中的文档一致
- [ ] 语言风格（正式程度、术语使用）与目标文件夹中的文档一致
- [ ] 编号方式与目标文件夹中的文档一致
- [ ] 表格/表单格式与目标文件夹中的配套表格一致
- [ ] 审批栏/签字栏格式与目标文件夹中的模板一致
- [ ] 文档间的引用关系准确
- [ ] 流程步骤与配套表格的字段匹配
- [ ] 审批权限与制度定义一致
